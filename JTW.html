How Sydneysiders travel to work

Randomised departure times

A while ago, i made this animated visualization which inspired me to look simulating and mapping people's travel patterns.

This map here shows the origins and destinations of work related trips across Metropolitan Sydney. The visualization simulates Sydneysiders travel patterns in the morning within a 60 minute window. Data was sourced from BTS JTW. The simulation of people's movements are done using R and excel

Syncronised arrival times

This is one of the first sketches for this project which shows an explosion of workers (around 2 million people) arriving at work all at the same time.

Synchronised departure times

On the other hand, this is what Sydney looks like if everyone left home for work at the same time.
